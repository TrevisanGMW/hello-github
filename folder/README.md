# Maya Scripts
Some useful scripts for Maya written in MEL and Python language.

## Organization
* `gt_tools`: gt_tools script package (MEL & Python).
* `mel-scripts`: contains scripts written in MEL
* `python-scripts`: contains scripts written in Python
